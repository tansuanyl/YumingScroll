# 社区行为准则

我们希望喻鸣绘卷社区保持专业、友善和建设性。所有参与者都应尊重不同经验、身份、观点和创作背景。

## 期望行为

- 围绕问题和代码提出具体、可验证的意见
- 接受善意反馈，并为错误负责
- 尊重隐私、版权、署名和保密要求
- 在分歧中保持克制，避免人身攻击
- 主动帮助新贡献者理解项目约束

## 不可接受行为

- 骚扰、歧视、威胁、侮辱或持续挑衅
- 未经允许公开他人的私人信息、凭证或未发布作品
- 提交违法、侵权、恶意或用于绕过安全控制的内容
- 滥用 Issue、PR、讨论区或维护者联系方式

维护者可以编辑或删除不当内容，关闭讨论，拒绝贡献，并对持续违规者限制参与。报告行为问题时请联系维护者并提供必要上下文，不要在公开讨论中扩大私人信息。
